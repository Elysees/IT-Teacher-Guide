## 学习建议

### 零基础学习路径
1. **第1-2周**: Python基础 → 数据类型 → 控制流
2. **第3-4周**: Python函数 → 面向对象 → 常用模块
3. **第5-6周**: C++基础 → 数据类型 → 控制流
4. **第7-8周**: C++函数 → 面向对象 → STL
5. **第9-10周**: 线性结构 → 树形结构
6. **第11-12周**: 图结构 → 哈希表
7. **第13-14周**: 排序算法 → 搜索算法 → 动态规划
8. **第15-16周**: 综合练习与项目实战
9. **第17-18周**: 总复习与模拟考试

### 有基础学习路径
- **Python基础**: 1-2天快速复习
- **C++基础**: 3-5天重点学习
- **数据结构**: 1-2周系统学习
- **算法专题**: 2-3周深入练习
- **模拟考试**: 1周冲刺
