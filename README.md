# 📚 信息技术教师招聘复习指南

> 面向信息技术教师招聘考试的Python与C++基础知识手册

## 🎯 项目简介

本项目为准备信息技术教师招聘考试的学习者提供系统的Python和C++基础知识复习资料。内容采用**手册式风格**，注重快速查找和实用性。

### 核心特色

- ✅ **手册式组织** - 结构化条目，便于快速查找
- ✅ **双语言对比** - Python与C++同步讲解
- ✅ **代码即文档** - 每个知识点配完整示例
- ✅ **算法实战** - 排序/搜索/动态规划等高频算法
- ✅ **教学友好** - FAQ、速查卡、教学指南

## 📖 内容结构

### Python教程 (6章)
- [01 - 基础入门](docs/python/01-basics.md) - 环境搭建、基本语法、变量运算符
- [02 - 数据类型](docs/python/02-data-types.md) - 数字、字符串、列表、字典、集合
- [03 - 控制流](docs/python/03-control-flow.md) - 条件语句、循环、逻辑运算
- [04 - 函数](docs/python/04-functions.md) - 函数定义、参数传递、Lambda、递归
- [05 - 面向对象](docs/python/05-oop.md) - 类对象、继承、多态、特殊方法
- [06 - 常用模块](docs/python/06-common-modules.md) - 文件、JSON、正则、时间

### C++教程 (6章)
- [01 - 基础入门](docs/cpp/01-basics.md) - 编译过程、命名空间、输入输出
- [02 - 数据类型](docs/cpp/02-data-types.md) - 指针、引用、数组、动态内存
- [03 - 控制流](docs/cpp/03-control-flow.md) - 条件、循环、位运算
- [04 - 函数](docs/cpp/04-functions.md) - 值传引用传、重载、递归、函数指针
- [05 - 面向对象](docs/cpp/05-oop.md) - 构造析构、继承、虚函数、操作符重载
- [06 - STL容器](docs/cpp/06-stl.md) - vector、map、set、算法、智能指针

### 算法专题 (3章)
- [排序算法](docs/algorithm/sorting.md) - 冒泡/选择/插入/快排/归并/堆排序
- [搜索算法](docs/algorithm/searching.md) - 线性/二分/跳跃/指数搜索
- [动态规划](docs/algorithm/dynamic-programming.md) - 斐波那契/背包/LCS/硬币/LIS

## 🚀 快速开始

### 在线阅读
访问 GitHub Pages: [https://Elysees.github.io/IT-Teacher-Guide/](https://Elysees.github.io/IT-Teacher-Guide/)

### 本地运行

```bash
# 克隆仓库
git clone https://github.com/Elysees/IT-Teacher-Guide.git
cd IT-Teacher-Guide

# 安装依赖 (需要Ruby和Bundler)
bundle install

# 启动本地服务器
bundle exec jekyll serve

# 访问 http://localhost:4000
```
